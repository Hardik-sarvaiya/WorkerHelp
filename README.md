# 👷 WorkerHelp

A full-stack MERN application that connects skilled workers with customers through a secure and user-friendly platform.

---

## 🚀 Overview

WorkerHelp helps customers find skilled workers while allowing workers to showcase their skills, experience, and contact information. The platform simplifies hiring and improves visibility for local professionals.

---

## ✨ Features

### 👷 Worker
- Register & Login
- Email Verification
- Create & Update Profile
- Add Skills & Experience
- Upload Profile Picture
- Manage Personal Information

### 👤 Customer
- Register & Login
- Browse Worker Profiles
- Search Workers
- View Worker Details

### 🔐 Authentication
- JWT Authentication
- Password Encryption
- Email Verification
- Protected Routes

### 📱 Responsive UI
- Mobile Friendly
- Tablet Friendly
- Desktop Friendly
